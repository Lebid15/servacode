# Phase 4.1 — Admin Sidebar RTL Fix

## المشكلة
في لوحة الأدمن، زر السايدبار كان يعمل في الواجهة الإنكليزية، لكنه لا يفتح بشكل صحيح في الواجهة العربية.

## السبب
كود السايدبار كان يعتمد على `start-0` و `rtl:translate-x-full` لإدارة موضع السايدبار وتحريكه. هذا جعل سلوك القائمة المتحركة مرتبطًا بتوليد Tailwind للـ RTL variant واتجاه المستند، مما سبب عدم ظهور السايدبار عند اللغة العربية على بعض الحالات.

## الإصلاح
تم جعل السلوك صريحًا حسب `locale`:

- العربية: السايدبار يثبت يمين الشاشة ويختفي إلى اليمين.
- الإنكليزية: السايدبار يثبت يسار الشاشة ويختفي إلى اليسار.

## الملف المعدل

- `admin_panel/src/shared/design-system/layout/AdminShell.tsx`

## نتيجة الإصلاح

- `/ar/admin/...`: زر السايدبار يفتح القائمة من اليمين.
- `/en/admin/...`: زر السايدبار يفتح القائمة من اليسار.
- لم يتم تغيير منطق الباكند أو API أو بنية الصفحات.
