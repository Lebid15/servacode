# Phase 13.4 — Initial Demo Content Seed

## الهدف

إضافة محتوى مبدئي احترافي لصفحات:

- تطبيقاتنا
- أنظمتنا
- أعمالنا

المحتوى لا يظهر للزائر على أنه تجريبي أو مؤقت، لكنه محفوظ في قاعدة البيانات كعناصر عادية يمكن تعديلها أو إخفاؤها أو حذفها من لوحة التحكم.

## السكربت الجديد

```powershell
cd D:\platform\backend
.\.venv\Scripts\activate
python scripts\seed_demo_content.py
```

## ماذا يضيف؟

### تطبيقاتنا

- منظّم المهام الذكي
- مولّد الفواتير
- محوّل الملفات الخدمي

### أنظمتنا

- نظام إدارة العملاء
- نظام إدارة المحتوى
- نظام إدارة العمليات

### أعمالنا

- واجهة شركة تقنية متعددة اللغات
- لوحة تحكم لإدارة المحتوى
- منصة خدمات رقمية

## قواعد الأمان

- السكربت لا يكرر البيانات عند تشغيله أكثر من مرة.
- يبحث عن `slug_ar` و `slug_en` قبل الإنشاء.
- إذا حذف المستخدم عنصرًا حذفًا ناعمًا من لوحة التحكم، لا يعيد السكربت إنشاءه تلقائيًا.
- المحتوى لا يستخدم أسماء شركات حقيقية ولا يدّعي مشاريع فعلية لعملاء.
- السكربت يفعّل أقسام `apps/products/portfolio/services/cta` في إعدادات الموقع حتى تظهر الصفحات مباشرة.

## الفائدة

بعد تشغيل السكربت، يصبح الموقع ممتلئًا بمحتوى أولي جيد بدل الصفحات الفارغة، ومع ذلك يبقى كل شيء قابلًا للإدارة من لوحة التحكم.
