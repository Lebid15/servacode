# Logo & Language Compact Hotfix

تعديل سريع على هيدر الموقع العام و زر اللغة.

## التعديلات

- زر اللغة أصبح أيقونة كرة أرضية فقط بدون AR/EN داخل الهيدر.
- ضبط حجم اسم الشركة والوصف القصير ليتناسبا مع الشعار الكبير.
- تقليل ارتفاع الهيدر بعد تكبير الشعار حتى لا يأخذ مساحة زائدة من الشاشة.
- إبقاء الشعار كبيرًا وواضحًا، لكن بدون ضغط النصوص أو عناصر التنقل.
