# Phase 13.4.6 — Light Theme Premium Polish

## الهدف
رفع جودة الثيم الفاتح حتى لا يظهر بدائيًا أو مسطحًا، مع الحفاظ على الثيم الداكن كما هو.

## ما تم

- إعادة بناء `public_site/src/app/globals.css` بشكل نظيف وإزالة التكرارات القديمة.
- تحويل الثيم الفاتح `emerald-luxury` إلى هوية فاتحة فاخرة: Ivory / Sapphire / Emerald.
- تحسين الخلفيات، الكروت، الظلال، الحدود، الحقول، وحالات التركيز.
- تحسين `glass-panel` و `premium-border` لتعمل بشكل احترافي على الثيم الفاتح.
- تحسين لون الحقول في النماذج ضمن الثيم الفاتح بدون التأثير على الثيم الداكن.
- تحسين ثيم لوحة التحكم الفاتح حتى لا يبقى أبيضًا مسطحًا.

## ملاحظات

- لم يتم تغيير المسارات أو منطق الباكند.
- التعديل مركزي عبر CSS variables، لذلك ينعكس على كل الصفحات العامة.
- بقي اسم الثيم في الكود `emerald-luxury` حتى لا تنكسر إعدادات قاعدة البيانات والباكند.
