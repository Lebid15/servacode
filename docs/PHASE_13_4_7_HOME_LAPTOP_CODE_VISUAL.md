# Phase 13.4.7 — Home Laptop Code Visual

## الهدف
إضافة عنصر بصري قوي في الصفحة الرئيسية يعرض شاشة لابتوب بداخلها كود برمجي يُكتب تدريجيًا، ليعطي انطباعًا تقنيًا احترافيًا بدل المشهد العام السابق.

## ما تم
- إضافة مكون جديد `HomeLaptopCodeVisual` داخل الموقع العام.
- استبدال المشهد البصري في Hero الصفحة الرئيسية بهذا المكون.
- تصميم مربع بصري متجاوب `aspect-square` يحتوي لابتوب وشاشة كود.
- إضافة تأثير كتابة CSS-only للكود داخل الشاشة.
- الحفاظ على دعم RTL/LTR دون كسر اتجاه الكود داخل الشاشة.
- استخدام ألوان وثيمات النظام المركزي عبر CSS variables.
- دعم الثيم الداكن والفاتح دون ألوان ثابتة خارج النظام.

## الملفات المعدلة
- `public_site/src/shared/public/components/HomeLaptopCodeVisual.tsx`
- `public_site/src/shared/public/components/PublicHero.tsx`
- `public_site/src/app/globals.css`

## ملاحظات
- لم يتم تغيير أي API أو مسار.
- العنصر يعمل كواجهة بصرية فقط ولا يؤثر على البيانات الديناميكية.
- الكود داخل الشاشة رمزي وتسويقي وليس كودًا تنفيذيًا.
