# Phase 13.1 — Page Naming & Navigation Refinement

## الهدف
توحيد أسماء الصفحات في الموقع العام ولوحة التحكم بحيث تظهر كواجهة شركة برمجيات احترافية، مع ترتيب "من نحن" قبل "تواصل معنا" كما تم اعتماده.

## التعديلات
- إعادة ترتيب روابط الشريط العلوي في الموقع العام إلى: الرئيسية، خدماتنا، تطبيقاتنا، أنظمتنا، أعمالنا، المقالات، من نحن، تواصل معنا، ابدأ مشروعك.
- إعادة تسمية Products إلى Systems / أنظمتنا دون تغيير المسار البرمجي `/products` للحفاظ على استقرار الروابط والـ API.
- إعادة تسمية Portfolio إلى Work / أعمالنا دون تغيير المسار البرمجي `/portfolio`.
- إعادة تسمية Blog إلى Insights / المقالات دون تغيير المسار البرمجي `/blog`.
- إعادة تسمية Quote Requests إلى Project Requests / طلبات المشاريع.
- إعادة تسمية Downloads إلى Download Center / مركز التحميل.
- إعادة تسمية Support إلى Support Center / مركز الدعم.
- إعادة تسمية Settings في لوحة التحكم إلى Company Identity / هوية الشركة.
- تحديث عناوين الوحدات، أوصافها، إحصائيات لوحة القيادة، والفوتر.

## قرار مهم
تم تغيير الأسماء الظاهرة فقط، ولم يتم تغيير المسارات البرمجية الحالية مثل `/products` و`/portfolio` و`/blog` حتى لا تنكسر الروابط أو استدعاءات الـ API.
