# Phase 15.7 — Professional Services Backend Seed

## الهدف

إضافة قائمة خدمات احترافية وشاملة داخل الباكند حتى لا يتم إدخال خدمات الشركة يدويًا واحدة واحدة من لوحة التحكم.

## الملف المضاف

```text
backend/scripts/seed_professional_services.py
```

## عدد الخدمات

تم تجهيز 21 خدمة احترافية قابلة للإدارة من لوحة التحكم، مع:

- عنوان عربي وإنجليزي.
- slug ثابت.
- وصف مختصر.
- وصف تفصيلي.
- SEO title / description.
- أيقونة متوافقة مع الموقع العام.
- ترتيب ظهور.
- 4 ميزات لكل خدمة.
- extra_data لتحديد مصدر البيانات والتصنيف.

## الخدمات المضافة

1. تطوير مواقع إلكترونية احترافية.
2. تطوير تطبيقات ويب تفاعلية.
3. تطوير لوحات تحكم وإدارة محتوى.
4. تطوير الأنظمة الإدارية المخصصة.
5. تطوير تطبيقات سطح المكتب Windows.
6. تطوير تطبيقات موبايل.
7. تحويل الأفكار إلى منتجات رقمية.
8. المتاجر الإلكترونية وحلول البيع.
9. أنظمة الحجز والمواعيد.
10. منيو إلكتروني وأنظمة طلب داخل المطعم.
11. أنظمة إدارة العملاء CRM.
12. ربط الأنظمة وواجهات API.
13. أتمتة الأعمال وسير العمل.
14. تصميم واجهات وتجربة المستخدم UI/UX.
15. برمجة الباكند وقواعد البيانات.
16. إضافة ميزات الذكاء الاصطناعي.
17. الهوية البصرية والبراند الرقمي.
18. صفحات هبوط وإعلانات رقمية.
19. تحسين وإعادة هيكلة المشاريع القائمة.
20. تجهيز النشر والاستضافة لاحقًا.
21. الصيانة والدعم التقني المستمر.

## طريقة التشغيل

من داخل مجلد الباكند:

```powershell
cd D:\platform\backend
.\.venv\Scripts\activate
python scripts\seed_professional_services.py
```

ثم افتح لوحة التحكم:

```text
http://127.0.0.1:3001/ar/admin/services
```

وافتح الموقع العام:

```text
http://127.0.0.1:3000/ar/services
```

## خيارات إضافية

عدم تعديل الخدمات الموجودة:

```powershell
python scripts\seed_professional_services.py --skip-existing
```

إضافتها مخفية / غير مفعلة:

```powershell
python scripts\seed_professional_services.py --inactive
```

## ملاحظات هندسية

- السكربت idempotent ويمكن تشغيله أكثر من مرة.
- يبحث عن الخدمة بواسطة `slug_ar` أو `slug_en`.
- إذا كانت الخدمة محذوفة soft-delete يتم إرجاعها وتفعيلها.
- يزامن الميزات الأربع لكل خدمة.
- الخدمات تبقى قابلة للتعديل من لوحة التحكم بعد إنشائها.
